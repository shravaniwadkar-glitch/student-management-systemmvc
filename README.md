## Advanced Student Management System

 # Overview
A full-screen Java Swing based Student Management System with MySQL integration.

# Features
- Admin login authentication
- Add / Update / Delete student records
- Search student by ID
- Full GUI dashboard
- JDBC database integration
- Validation and exception handling

# Technologies Used
- Java
- Swing
- JDBC
- MySQL
- Eclipse IDE

# Database Setup
1. Create database `studentdb`
2. Run SQL script
3. Add admin credentials
